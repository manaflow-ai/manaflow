## Web app
Open the cmux web app in your browser and sign in. The web app runs tasks in cloud workspaces.

## Desktop app (Electron)
Download and install the desktop app from the cmux website, then sign in.

Desktop app benefits:
- Local workspaces on your machine.
- Cloud workspaces in the same UI.

Local workspaces are available only in the desktop app.

## Local requirements
Local workspaces require:
- Docker (running)
- Git
